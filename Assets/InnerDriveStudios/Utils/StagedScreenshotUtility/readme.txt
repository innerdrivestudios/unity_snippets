Description:
- Simple utility to take screenshots of objects in your Scene and turn them into Sprites.

Usage:
- Image you are building a tower defense game and you would like to create images of the towers players can buy. This utility allows you to simple place all assets for which you would like to create a sprite in the ScreenShotUtility Scene and "Play" it, converting all those 3d assets to 2d (transparent) sprites automatically.

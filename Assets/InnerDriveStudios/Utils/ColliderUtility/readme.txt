Description:
- ColliderUtility assists you in easily adding bounding colliders to a hierarchy of objects.

For example: 
- you have a house made up of 30 primitives (Cube, Spheres, etc) but all you need for your scene is one bounding capsule collider.

Usage:
- Utility/Inner Drive Studios/Colliders/Remove all colliders from selected objects
- Utility/Inner Drive Studios/Colliders/Add bounding ... collider

Options:
- Make sure to check/experiment with the different bounding options (inside, outside, midway).
- There is no "one size fits all", whatever works best depends on your specific situation.

Version history:
- Version 1 - Base version
- Version 2 - Added demo / Fixed namespaces
- Version 3 - Added the option to add bounding box/capsule/sphere colliders
- Vection 4 - Rewrote the code to improve the bounds matching
- Version 5 - Added Fitmode options for radius and height: Inside, Outside, Midway

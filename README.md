# Unity Snippets
Repository with some handy unity snippets and tools, use at your own risk :).

## Tools:
1. Circlelizer - quickly position elements in a circle
2. ColliderUtility - remove all colliders from objects and re-add bounding box, sphere or capsule colliders
3. MissingComponentUtility - remove all missing components from GameObjects and Prefabs automatically
4. Notes - add documentation notes directly to your game objects
5. PivotPointUtility - like 'Center on Children' but then on the bottom center ;)
6. PrefabReplacer - replaces any object in your scene with a prefab selected from a set
7. StagedScreenshotUtility - stage your prefabs and turn them into transparent 2D sprites automatically
8. Attributes:
	- HelpBoxAttribute -> 		add an explanation to a property visible in the inspector
	- ReadOnlyAttribute -> 		add a value in the inspector that is visible but not editable
9. Properties:
	- MinMaxRangeProperty ->	add a range of values, instead of using Vector2.xy for that
